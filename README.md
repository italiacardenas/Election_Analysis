# Election Audit

## Overview of Election Audit
This is an election audit of tabulated results completed by the Colorado Board of ELections for the U.S. Congressional Precinct in Colorado to certify this U.S. Congressional race. The three voting methods taken into account were mail-in ballots, punch cards, and direct recording electronic. The report contains the total number of votes cast, the total number of votes for each candidate, the percentage of votes for each candidate, the winner of the election based on popular vote. 

The process was automated with Python 3.7.6. The automation was successful and may be used to audit other congressional districts, senatorial districts and local elections.
